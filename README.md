# epigoni-docs

Situs Dokumentasi Epigoni.

## Prasyarat

- NodeJS
- Yarn (rekomendasi) atau NPM
- Git

## Instal

menginstal dependensi

`yarn install`

atau

`atau npm install`

## Scripts

Menjalankan situs di lokal:

`yarn docs:dev`

atau

`npm docs:dev`

Untuk menghasilkan aset statis:

`yarn docs:build`

atau

`npm docs:build`
