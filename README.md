# epigoni-docs

Situs Dokumentasi Epigoni.

## Prasyarat

- NodeJS
- Yarn (rekomendasi) atau NPM
- Git

## Instal

Menginstal dependensi:

```bash
yarn install
```

atau

```bash
npm install
```

## Scripts

Menjalankan situs di lokal:

```bash
yarn docs:dev
```

atau

```bash
npm docs:dev
```

Untuk menghasilkan aset statis:

```bash
yarn docs:build
```

atau

```bash
npm docs:build
```
